# Packages

## Required for the tool

- Python3 and Pip3
- Ruby-full
- lsb-core
- build-essential
- graphicsmagick
- xdg-utils

Other packages will be installed when running specific scripts.

## Optional packages include

- jq
- clang-format
- groff
- docker & docker-compose
- virtualenv
- rubocop
- ccrypt
- openssl
- zip
- unzip

and more

Check [TESTED_ON.md](TESTED_ON.md) to see versions.
